React useEffect cleanup

React’s useEffect hook is a super hook to run side effects. You might be wondering what kind of side effects we could be talking about?

在 React 中 useEffect 是一个 super hook 钩子，帮助组件运行副作用，你可能会好奇是什么样的副作用，下面有几个例子：

  • Change the document title based on rendered values
  • Get or set values from local storage
  • Run analytics events
  • Show some greeting based on time of day
  • Focus on a form field after load>
  • 基于渲染的值，改变网页的标题
  • 通过本地存储获取或改变变量的值
  • 运行分析事件
  • 基于当前时间显示问好信息
  • 当加载完成后，聚焦到表格

The basic use of useEffect

useEffect 基本的用法，大多数情况下，只要按照预期使用该钩子已经足够了，比如：

useEffect(() => {
  document.title = `The page is loaded`;
});

也可以设置依赖（当依赖发生改变时，触发钩子）

useEffect(() => {
  document.title = `The page ${title} is loaded`;
}, [title]);

Cleaning up useEffect

useEffect 钩子带有 cleanup 函数，你可能并不经常使用这个功能，但它在很多地方可以派上用场。

为了调用 cleanup 清理函数，只需要简单地添加一个 return 函数：

useEffect(() => {
  // Your effect

  return () => {
    // Cleanup
  };
}, []);

cleanup 清理函数可以防止内存泄漏以及去除不需要的东西，比如：

  • 清理订阅
  • 清理 modal 对话框
  • 清理事件监听
  • 清理 timeout 事件

下面是一个例子，在特定时间后添加一些事件：

const [show, setShow] = useState(false);
useEffect(() => {
  let timer = setTimeout(() => setShow(true), 3000);
}, []);

这会在内存中创建一个 timeout 事件，如果能清理这个事件就好了，比如使用 cleanup：

useEffect(() => {
  let timer = setTimeout(() => setShow(true), 3000);
  return () => {
    clearTimeout(timer);
  };
}, []);

另一个例子是保持轮询的 web socket 套接字调用的轮询 polling：

useEffect(() => {
  let ws = new WebSocket('wss://ws.your-websocket/');

  ws.onmessage = (msg) => {
    // Do something with the message
  };

  return () => ws.close();
}, []);

这个例子中，我们打开了 WebSocket 链接，然后使用 cleanup 清理函数关闭链接。

另一种你可以使用 cleanup 清理函数的情况是你可以用来追踪 modal 对话框关闭事件，举个例子：

假设我们有一个对话框，在这个 modal 组件中，我们添加一个 useEffect，并且在 cleanup 清理时添加一个触发函数。 通过这种方式，我们可以捕捉每个事件。

此时，当另一个组件被调用时候，modal 对话框会被触发，所以我们没有办法告诉用户什么时候去关闭它（除了直接关闭应用外）

对于这种情况，我们可以添加一个 cleanup 函数触发 effect

useEffect(() => {
  return () => {
    trackModalClose();
  };
}, []);

通过这种方式，我们有了一个可靠的方法，来追踪什么时候唤醒 modal 对话框关闭事件，并且你可以在应用或对话框被关闭时，添加一个确认。

Conclusion

总结来说，这篇文章粗略介绍了 useEffect 中 return () => {} 即 cleanup 函数的作用，正如其名，可以用于清理一些订阅或事件，比如对话框关闭。
