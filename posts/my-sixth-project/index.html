使用 Python 做高数题目

高数工具

SymPy

sympy 是一个 Python 的科学计算库，用一套强大的符号计算体系完成诸如多项式求值、求极限、解方程、求积分、微分方程、级数展开、矩阵运算等等计算问题

安装：

通过在 powershell 或是 cmd 使用 pip install sympy 安装 sympy 模块

官方教程：

https://docs.sympy.org/latest/tutorial/index.html

表达式与表达式求值
from sympy import *

x = Symbol('x')
y = Symbol('y')
fx = x * x + y * y
result = fx.evalf(subs={x: 3, y: 4})
print(result)

函数方程求解
from sympy import *

x = Symbol('x')
y = Symbol('y')
fx = x * 3 + 9
print(solve(fx, x))

极限

$$ \lim\limits*{x\to0} \frac{sinx}{x} = 1 $$ $$ \lim\limits*{x\to0} (1+x)^{\frac{1}{x}} = e $$ $$ \lim\limits_{x\to0} (1+\frac{1}{x})^{x} = e $$

from sympy import *

x = Symbol('x')
f1 = sin(x) / x
f2 = (1 + x)**(1/x)
f3 = (1 + 1/x)**x
# limit(expression, symbol, the value toward which z tends)
lim1 = limit(f1, x, 0)
lim2 = limit(f2, x, 0)
lim3 = limit(f3, x, oo)
print(lim1, lim2, lim3)

线性代数工具

Numpy

Numpy 是向量计算的基础库，Numpy 是一个很重要、使用很多的科学计算包。在 Python 领域里有很多的与科学计算相关的包都或多或少地都使用到了 Numpy 这个模块包，例如著名的大数据包 pandas 就是基于 numpy

安装：

通过在 powershell 或是 cmd 使用 pip install numpy 安装 numpy 模块

官方教程：

https://numpy.org/doc/stable/

求方程组的解

$ \begin{bmatrix} 1 & 1 & 1 \\ 0 & 2 & 5 \\ 2 & 5 & -1 \end{bmatrix} $ $ \begin{bmatrix} x \\ y \\ z \end{bmatrix} $ $ = $ $ \begin{bmatrix} 6 \\ -4 \\ 27 \end{bmatrix} $

from numpy import *
import numpy as np

a = np.array([[1, 1, 1], [0, 2, 5], [2, 5, -1]])
b = np.array([[6], [-4], [27]])
x = np.linalg.solve(a, b)
print(x)
print(np.allclose(np.dot(a, x), b))

其中 allclose 函数返回 True 如果两个向量在容差内是相等的

矩阵乘法

$ \begin{bmatrix} 1 & 2 & 3 \\ 4 & 5 & 6 \end{bmatrix} $ $ \begin{bmatrix} 2 & 3 & 4 \\ 5 & 6 & 7 \\ 8 & 9 & 10 \end{bmatrix} $ $ = $ $ \begin{bmatrix} 36 & 42 & 48 \\ 81 & 96 & 111 \end{bmatrix} $

from numpy import *
import numpy as np
a = np.arange(1, 7).reshape([2, 3])
b = np.arange(2, 11).reshape([3, 3])
print(a)
print(b)
print(np.matmul(a, b))
print(np.dot(a, b))

其中 matmul 和 dot 函数都能实现矩阵乘法

用 Python 做高数/线代作业

高数

求极限

$$ \lim\limits_{x\to0^+} (\frac{1}{\sqrt{x}}) ^{tanx} = ? $$

from sympy import *

x = Symbol('x')
f1 = (1 / sqrt(x)) ** tan(x)
lim1 = limit(f1, x, 0, dir="+")
print(lim1)

求导数

$$ y = e^x cos x, 求 d^2y $$

from sympy import *

x = Symbol('x')
y = exp(-x)*cos(x)
dy = diff(y, x)
d2y = diff(dy, x)
print(d2y)

线性代数

$$ A = \begin{bmatrix} 0 & 1 & 3 \\ 1 & -1 & 0 \\ -1 & 2 & 1 \end{bmatrix} 的逆矩阵 A^{-1} $$

import numpy as np

x = np.array([[0, 1, 3], [1, - 1, 0], [-1, 2, 1]])
print(x)
y = np.linalg.inv(x)
print(y)
print(np.dot(x, y))

$$ 方程组\\ x_1 + x_2 + x_3 + x_4 = 1 \\ 2x_1 + 3x_2 + 4x_3 + 5x_4 = 5 \\ 4x_1 + 9x_2 + 16x_3 + 25x_4 = 25 \\ 8x_1 + 27x_2 + 64x_3 + 125x_4 = 125 \\ 有解() $$

import numpy as np

a = np.array([[1,1,1,1],[2,3,4,5],[4,9,16,25],[8,27,64,125]])
b = np.array([[1], [5],[25], [125]])
x = np.linalg.solve(a, b)
print(x)
print(np.allclose(np.dot(a, x), b))
