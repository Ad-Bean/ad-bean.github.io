游戏介绍

你是超级英雄——蜘蛛侠，在未来的某一天醒来时，你突然发现整座城市被颠倒了！

作为城市的友好邻居，使用你的蛛丝发射器，拯救这座城市吧！

操作方法

使用鼠标左键发射蛛丝，并且按住使自己荡起来，你的目标是在敌人的魔音下，尽可能地生存下去，直到援军赶来！

注意：请适当调节背景音乐大小！

制作开始页面

创建好项目之后，引入眼帘的就是这个页面，左侧是属性栏，中间是操作页面，右侧是目录，我们先在右侧目录栏中将 Layout 1 右键改名 Rename 为 Start 意为开始页面，并着手制作开始页面。

  1. 双击页面空白，选择待创建对象的类型，创建一个新的 Sprite 对象以及一个鼠标对象

  2. 为了使开始页面能顺利跳转到游戏页面，将背景设置为 Sprite 对象，左键单击要放置的位置，并将准备好的背景图拖入编辑页面

  3. 右键 Sprite 背景，将其设置为覆盖视图

  4. 新建一个 Layout 页面并进行跳转设置

  5. 对 Layout 和 Even Sheets 进行相应地命名。切换回 Start 页面的 Even Sheets，对背景设置跳转功能

  6. 双击空白处，选择 Mouse 即鼠标对应的事件，选择当左键按下时，并且添加对应的操作

如此，我们就完成了开始页面的制作，并且当左键开始页面的任何一处时，都能顺利地从开始页面跳转到游戏页面。

制作游戏页面

  1. 按照上述教程，我们在游戏页面也添加多个 Sprite，分别为背景、鼠标、蜘蛛侠、颠倒城市、蛛网等等。

  2. 在属性栏中给 Sprite 蜘蛛侠添加各种行为，并且在编辑页面中设置其 Image Point，即蛛丝发射点。

    Line of Sight 视线：测试一个对象的位置是否具有可观察到另一个对象的视野（让蛛丝不能透过颠倒城市）

    Physics 物理：给蜘蛛侠添加重力

    Destroy Outside Layout 出界销毁：添加销毁判定

  3. 为游戏页面添加事件，如当页面加载时设置重力，播放背景音乐，当人物出界时销毁并回退到开始页面，发射蛛丝等等行为。

总结

除了给精灵 Sprit 添加行为和设置事件以外，还可以使用 JavaScript 或者 Lua 脚本语言为你的游戏增添更多功能。

我在学习如何用 Construct3 制作游戏的过程中，也能稍微领会到一个游戏的诞生过程。从构思到拓展，再到真的实现，不论是用 Construct3 的可视化编程，还是在网上寻找或者自己制作各种游戏资源（人物，背景，音乐等等），总而言之制作游戏并不是一件容易的事情，尽管可视化编程和现在的许多游戏制作软件大大简化了这个过程。

但我认为如果想要一个更加完整的游戏，我还需要继续学习，学习游戏引擎 Unity、UE4 等等，学习 3D 模型的制作，学习游戏配乐等等，这是一个很长的旅途，希望自己能在未来能实现一个游戏梦！

注：游戏中一切资源均来源网络，部分经过个人加工，请勿用于商业用途
