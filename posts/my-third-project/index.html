  2. 画真值表验证与门功能

INPUT A(logic source) INPUT B(logic source) OUTPUT(led)
0 0 OFF(0)
0 1 OFF(0)
1 0 OFF(0)
1 1 RED(1)
  1. 选做题：
    • 如何用 XOR 门作非门。
    • 串联、并联三极管，结果是什么门？

  • 当两输入异或门的其中一个输入固定为 1，则当另一个输入为 0 时，输出为 1，另一个输入为 1 时，输出为 0，此时 XOR 门可以当做非门。

  • 串联、并联三极管，结果是什么门？

    INPUT A(logic source) INPUT B(logic source) OUTPUT(led)
    0 0 OFF(0)
    0 1 OFF(0)
    1 0 OFF(0)
    1 1 RED(1)

    INPUT A(logic source) INPUT B(logic source) OUTPUT(led)
    0 0 OFF(0)
    0 1 RED(1)
    1 0 OFF(1)
    1 1 RED(1)

综上： 三极管串联是与门，三极管并联是或门。
